# 🚧Mictory

Microphone + Victory = Mictory

跨平台私有开黑语音服务

[文档](https://smilecc.github.io/mictory/) | [在线 Demo](https://mictory.smilec.cc/)

## 快速开始

```bash
docker compose up
```
